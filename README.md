Generated project for **Northway Engineering**. This is a minimal, ready-to-deploy Next.js site with Tailwind CSS.

## Quick start (local)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   npm run start
   ```

## Deploy to Vercel
1. Push this repository to GitHub.
2. Go to https://vercel.com, import the repo, and deploy. Vercel auto-detects Next.js.
